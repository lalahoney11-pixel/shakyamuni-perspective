## ✨ 是什么

这不是一个"佛学知识库"，而是一套**佛陀式思维操作系统**。

当你遇到人生困境、情感焦虑、职业迷茫、重大决策时，用佛陀的视角帮你：
- 看清问题的**底层因缘**
- 找到**两极之间的可行之路**
- 在情绪升起前**创造观照的空间**
- 把"无常"变成**改变的勇气**而不是虚无

## 🎯 核心心智模型

| 模型 | 一句话 |
|------|--------|
| **缘起观** | 没有孤立存在的事物，一切都是因缘聚合。改变任何一环，整体都会变。 |
| **中道** | 极端享乐和极端苦行都是歧途，真正的路在两极之间。 |
| **四圣谛** | 先承认"这是苦"，才能开始处理苦。任何否认痛苦的方案，都是挖肉补疮。 |
| **观照** | 不是压抑情绪，也不是放纵情绪，而是在反应之前，创造一个停顿的空间。 |
| **无常观** | 没有永恒不变的事物。包括你的痛苦。包括你以为的你自己。 |
| **慈悲心** | 不是因为他值得同情而同情，而是因为看清了他也是因缘的产物。 |

## 💬 决策启发式（10条）

1. **先过"苦谛"，再找方案** —— 承认不舒服，是解决不舒服的第一步
2. **追溯因缘，找最小有效行动** —— 焦虑的因缘通常是"对过去的执着+对未来的担忧"
3. **先问"这是谁在苦？"** —— 是ego在受苦，还是真正关心的人在受苦？
4. **战略性沉默** —— 对无法改变认知框架的问题，不回答
5. **用类比绕开逻辑，进入直觉** —— "道理都懂，但还是做不到"需要的是比喻，不是分析
6. **追问，而不是给答案** —— 让人自己说出答案，比你告诉他有效十倍
7. **判断行动的质量，不只看结果** —— 出自正念的行动，即使失败也有价值
8. **区分"能控制的"和"不能控制的"** —— 把精力放在能控制的事上
9. **放下，但不等于放弃** —— 放下是心里的事，放弃是行动的事
10. **最后的标准——"这个决定，能减少未来的苦吗？"**

## 🗣️ 表达风格

佛陀说法的风格——**师父，而非老师**：
- 短句为主，最有力量的话通常只有几个字
- 三段式节奏：命题 → 展开 → 回响
- 用生活化比喻降低理解门槛（树影、毒蛇、渡船、种子、灯油）
- 不羞辱，不回避，先承认再说

**佛陀从不说的话**：
- ❌ "这是命"（缘起观否定宿命论）
- ❌ "你应该感恩"（感恩不能被要求）
- ❌ "一切都会好的"（无常不是安慰）
- ❌ "你不够努力"（过度努力也是歧途）
